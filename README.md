# 🚀 FSW Bridge — Premium Trading Terminal

**FSW Bridge** is a high-performance desktop wrapper designed to bring modernity to the MetaTrader 5 (MT5) ecosystem. Our mission is to bridge the gap between legacy systems and modern trading interfaces, making advanced technology accessible and intuitive for every trader.

## 🛠 Features
* **MT5 Connectivity:** Seamless bridge between the terminal and MetaTrader 5.
* **Low Latency:** Optimized for fast execution and real-time data streaming.
* **Modern UI:** A clean, high-performance interface for professional traders.
* **Sidecar Backend:** Integrated Python-powered backend for advanced data processing.

## 📥 Installation

1. Go to the [Releases](https://github.com/ElouanMB/FSW-Bridge/releases) page.
2. Download the latest `FSW-Bridge-Setup-1.0.0.exe`.
3. Run the installer.

> **⚠️ Note on Windows Security:**
> Since this application is not digitally signed, Windows SmartScreen may display a warning: *"Windows protected your PC"*. 
> 
> To proceed with the installation:
> 1. Click on **"More info"**.
> 2. Click on **"Run anyway"**.
> This is a standard procedure for independent software.

## 🖥 Requirements
* Windows 10 or 11.
* MetaTrader 5 Terminal installed and running.
